# Ekspektasi Aset Examination

Folder ini berisi gambar hasil pemeriksaan yang muncul setelah user request examination dan proses loading selesai.

File yang perlu disiapkan:
- obgyn-ultrasound-early-pregnancy.png: untuk case "Perdarahan Kehamilan Muda". Tampilkan USG obstetri awal dengan hasil yang sesuai: Kantung kehamilan intrauterin terlihat, denyut belum jelas pada simulasi ini, tidak tampak cairan bebas.

Ekspektasi gambar:
Gambar tidak harus fotorealistik, tetapi harus mudah dikenali sebagai hasil pemeriksaan medis. Gunakan gaya visual yang konsisten dengan PixelAid, bersih, tidak menakutkan, dan cukup jelas agar user memahami mengapa hasil tersebut penting dalam case.
